# Deprecated Documents

이 디렉토리는 현재 `artifact-handoff`의 resolver-service 기준선과 맞지 않는 이전 문서들을 보관한다.

원칙:

- 이 디렉토리의 문서는 historical reference로만 사용한다.
- 새 구현 판단의 기준 문서로 사용하지 않는다.
- 현재 기준선은 상위 `docs/`의 `PRODUCT_IMPLEMENTATION_DESIGN`, `ARCHITECTURE`, `DOMAIN_MODEL`, `PLACEMENT_AND_FALLBACK_POLICY`, `DRAGONFLY_ADAPTER_SPEC`, `PHASE1_RESOLVER_STATUS`를 따른다.

이동 이유:

- 일부 문서는 controller-oriented wording을 강하게 유지하고 있었다.
- 일부 문서는 현재 resolver-service 아키텍처와 디렉토리 구조를 잘못 시사했다.
- 다음 스프린트 진행 전, 현재 기준선과 historical 문서를 분리할 필요가 있었다.
